                                          ----VIBE VOICE----



This project uses speech emotion recognition (SER) to detect a user's emotional state from their speech and 
then provides personalized suggestions (like listening to music or playing games) to improve their mood.

--> Overview:
- **SER FOULDER:** Processes audio input and classifies emotions (happy, sad, angry).
- **Suggestions Foulder:** Maps the detected emotion to a list of recommended activities.

-->Steps To Run:
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app/main.py` or `python app/web_page.py`

**make sure u have required extenstion (ffmpeg)
